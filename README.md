# 🎮 Gaming Website

A modern and responsive **Gaming Website UI** built using **HTML, CSS, and JavaScript**.  
This project showcases a stylish gaming interface with a navigation menu, sections for story, products, and gaming news.

It is designed as a **frontend practice project** to improve web development skills and create a visually appealing gaming themed website.

---

## 🚀 Features

✨ Modern Gaming UI Design  
📱 Fully Responsive Layout  
🎮 Interactive Navigation Menu  
📰 Sections for Game Story, Products, and News  
⚡ Smooth and Clean User Interface  

---

## 🛠️ Technologies Used

- **HTML5** – Website structure  
- **CSS3** – Styling and layout  
- **JavaScript** – Interactive functionality  
- **Remix Icons** – Modern icons
